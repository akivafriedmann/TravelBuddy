2. The Palette: "Charcoal" vs. "Pitch Black" ⚫
In your screenshot, the background looks like pure black (#000000).

The Problem: Pure black can feel harsh and cheap.

The Premium Fix: Use "Dark Grey/Charcoal" (#121212 or #1A1A1A) for the background. This allows you to use shadows (which are invisible on pure black) to create depth.

CSS Tweak:

CSS
body.dark-mode {
  background-color: #121212; /* Soft premium dark */
  color: #E0E0E0; /* Off-white text is easier to read than pure white */
}

/* Make cards slightly lighter to "float" */
.dark-mode .restaurant-card {
  background-color: #1E1E1E;
  box-shadow: 0 4px 20px rgba(0,0,0,0.5); /* Deep shadow */
}
