 * Fix Input Focus Logic (JavaScript):
   * In your createListBtn click listener, prevent the click from bubbling down to the map.

const createListBtn = document.getElementById('create-list-btn'); // or whatever ID you used
const newListNameInput = document.getElementById('new-list-name');

if (createListBtn) {
    createListBtn.addEventListener('click', (e) => {
        e.preventDefault(); // Stop form submit
        e.stopPropagation(); // Stop map click

        const name = newListNameInput.value;
        if (name) {
            createNewList(name);
        }
    });

Why this happens
 * Photos: Flexbox on mobile tries to "fit everything." If you have a lot of text reviews, it shrinks the image to zero to make room. flex-shrink: 0 stops this.
 * Button: The map has a very high Z-Index (usually 100-1000). If your popup is 999, the map wins. Setting it to the "Max Integer" (2147483647) guarantees it wins.
